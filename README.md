# WhatsApp_Clone
A WhatsApp UI clone built using Flutter as part of an app development project. This project replicates the user interface of WhatsApp, including chat screens and calls.

##Features
- Chat screen with messages layout
- Calls screen UI
- Navigation and bottom tabs

## Tech-Stack
- Flutter(Dart)

## Future Improvements
- Implement real-time messaging
- Add authentication (Google, Phone number)
- Backend integration with Firebase
